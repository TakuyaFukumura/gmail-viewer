Gmail API設定手順

  1. Google Cloud Consoleにアクセス
  2. 新しいプロジェクトを作成
  3. プロジェクト名: gmail-viewer（任意の名前）

  1. Google Cloud Consoleの「APIとサービス」→「ライブラリ」を選択
  2. 「Gmail API」を検索
  3. 「Gmail API」を選択し「有効にする」をクリック

  1. 「APIとサービス」→「認証情報」を選択
  2. 「認証情報を作成」→「OAuth 2.0 クライアント ID」を選択
  3. アプリケーションの種類: 「ユーザーデータ」を選択
  4. リダイレクトURI: http://localhost:8080/oauth2/callback
  5. 作成後、クライアントIDとクライアントシークレットを控える

以下の環境変数を設定してアプリケーションを再起動してください：

Linux/macOS:
export GOOGLE_CLIENT_ID="あなたのクライアントID"
export GOOGLE_CLIENT_SECRET="あなたのクライアントシークレット"
export GOOGLE_REDIRECT_URI="http://localhost:8080/oauth2/callback"
Windows:
set GOOGLE_CLIENT_ID="あなたのクライアントID"
set GOOGLE_CLIENT_SECRET="あなたのクライアントシークレット"
set GOOGLE_REDIRECT_URI="http://localhost:8080/oauth2/callback"
注意: 環境変数設定後、アプリケーションを再起動する必要があります。
